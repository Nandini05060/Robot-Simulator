FLEET MANAGER SIMULATOR GUIDE

This portal serves as an interactive showcase of bluCursor's capabilities in **frontend system simulator modeling** and **enterprise user experience design**. Use the interactive phone to simulate operating autonomous logistics hardware in real-time:

1. Boot & Authentication Tap the splash screen inside the phone view, then log in using simulated credentials. Use operator@blucursor.com / password123, or use admin@blucursor.com / admin2024 to unlock administrative controls.
2. Operations Modules Navigate through the Delivery Logistics or Patrol Surveillance cards. Select floor maps A-E or patrol zones to inspect unit statuses, battery metrics, and paths.
3. Telemetry & Remote Overrides Tapping "Open Live Map & Control" launches real-time telemetry. Hold down the D-pad steering buttons (Up, Down, Left, Right) to manually drive the robot, adjust speeds, or hit the red E-STOP override.
4. Configurations & Dark Mode Open Settings to swap between simulated Light Theme and Dark Theme. You can also toggle Admin mode to simulate supervisor user profiles dynamically.
